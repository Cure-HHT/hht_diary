# Generated Reports

Canonical home for generated reports across the repo.

## Layout

- `reference/` — committed, human-reviewed artifacts that are generated by a
  script and then checked into git. Drift-checked in CI (CI regenerates and
  fails if the committed copy is stale). Example: `mobile-env-delta.md`, the
  controlled delta record for `DIARY-OPS-single-promotable-artifact/D`.
- `build/` — ephemeral CI build-time outputs (SBOM, coverage). Git-ignored;
  uploaded as workflow artifacts.

## Migrating existing reports

SBOM (`build-sponsor-ci.yml`), coverage (`lcov.info`), and elspais traceability
(`spec/_generated/`) predate this convention and live elsewhere. Migrating them
into `_reports/` is tracked as a follow-up ticket — do not fold it in ad hoc.
