Full Name
: A common name of the person associated with a **User Account** that
  uniquely identifies a specific individual in the context of the
  **Sponsor Portal**. A Full Name is **PII**.

Email Address
: A unique technical identifier used as a destination for system notifications
  and as a primary User ID for authentication.
: Reference Term: **Registered Notification Address**
: Reference Source: **ISO/IEC 24760-1**

ISO/IEC 24760-1
: Reference
: Title: IT Security and Privacy — A framework for identity management
: Version: ISO/IEC 24760-1:2019
: Effective Date: 2026-03-22
: URL: <https://www.iso.org>
